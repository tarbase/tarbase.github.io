What we do

We help customers build and deploy scalable cloud infrastructures using Amazon Web Services (AWS).

Network topology

Set up a best practices network topology in your AWS account.

Monitoring

Set up monitoring and alerting using AWS CloudWatch.

Docker Cluster

Deploy your Docker containers on top of the AWS EC2 Container Service (ECS).

Security

Set up security best practices for managing secrets, credentials, and servers.

Continuous Delivery

Set up an automated pipeline to build and deploy code using a self-hosted CI server (e.g. Jenkins) or cloud-based CI service (e.g. CircleCI, or Travis)

Relational Database

Run MySQL, Postgres, MariaDB, or Amazon Aurora on Amazon’s Relational Database Service (RDS).

Distributed Cache

Run Redis or Memcached clusters using Amazon’s ElastiCache Service.

Stateful Server

Set up a best-practices deployment of a single, stateful server on top of AWS.
